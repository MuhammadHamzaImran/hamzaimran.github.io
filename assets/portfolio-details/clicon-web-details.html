Clicon - A Modern E-Commerce Experience

Clicon is a modern e-commerce platform built to provide users with a seamless and engaging online shopping experience. The system was designed with performance and scalability in mind, combining a robust Django REST API backend with a fast and dynamic React.js frontend.

The platform leverages PostgreSQL as the primary database, ensuring secure and efficient data handling for products, users, and orders. It was deployed on Vercel for frontend hosting and Google Cloud Run for backend microservices, providing a scalable and production-ready solution.

Key Features:

  • Modern UI/UX for smooth navigation
  • Product discovery with search and filtering
  • Seamless cart and checkout process
  • Scalable backend with Django REST APIs

This was a personal project that I developed independently in just 2 months. As a Full Stack Developer, I was responsible for the entire lifecycle of the project including backend API development, frontend implementation, database design, and deployment to production environments.
