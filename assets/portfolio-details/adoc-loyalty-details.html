Frontend Development for ADOC Loyalty Program App

The ADOC Loyalty App is a platform designed to enhance customer engagement and reward loyalty across multiple brands, including ADOC, CAT, Hush Puppies, and The North Face. The app features responsive UI components built using Flutter and integrates seamlessly with Firebase for efficient functionality.

The app supports language conversion to Spanish, catering to Spanish-speaking regions. Firebase integration includes Google Sign-In functionality and push notification services to enhance user engagement and app interactivity. These features ensure a localized, efficient, and connected user experience.

A key challenge was ensuring the app provided a user-friendly interface while managing real-time updates for promotions and reward points. Firebase cloud functions were utilized to handle backend interactions efficiently, delivering a seamless user experience.

The ADOC Loyalty App has become a valuable tool for Empresas ADOC to engage their customers across various regions, with a focus on creating scalable and visually appealing interfaces.
