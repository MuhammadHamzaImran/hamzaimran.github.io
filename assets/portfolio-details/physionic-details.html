UI Design and React Frontend Development for Physionic

Physionic is a user-friendly website developed to provide medical professionals and enthusiasts with a platform to access vital resources and insights about health and medicine. This project focused on creating an engaging and visually appealing interface to deliver information efficiently.

The UI was carefully crafted using Figma, prioritizing simplicity and ease of navigation for users. React was utilized to develop a responsive, interactive, and high-performing frontend, ensuring compatibility across devices and browsers.

The website features sections for blogs, research articles, and medical tools to aid users in staying informed about the latest developments in the medical field. The design emphasizes accessibility and an intuitive layout to enhance user experience.

This project refined my expertise in React development and frontend design while allowing me to contribute to a platform that serves as a valuable resource for the medical community.
